# 番茄短故事爆款规则库

## 核心目标

不是文学性，是完读率、追读率、停留时长、情绪上瘾。

## 官方口径

- 番茄小说短故事是一篇完结的小说体裁，不是短剧分集剧本。
- 篇幅范围：6000-80000字，10000-30000字更适合短故事爆发。
- 核心特征：节奏紧凑、剧情跌宕起伏、感情张力强、沉浸感浓烈。
- 叙事建议：优先第一人称，代入感更强。
- 发布形态：一篇文章完结，不按传统长篇章节追更。

## 硬规则

- 100字内必须出现冲突、背叛、危机、打脸、离婚、压迫、捉奸、死亡、追杀、破产、系统、重生之一。
- 300字内必须出现巨大钩子、身份反差、第一次反转、核心悬念。
- 每800-1200字至少出现一次打脸、爽点、反转、危机升级、暧昧升级、身份曝光、情绪爆发、极端偏爱、翻盘。
- 每个生成段落结尾必须留钩，但最终成稿不显示分段标题。

## 禁止项

禁止文学风、文青风、慢热、长铺垫、大量世界观、长环境描写、哲学思考、大段心理活动、无意义聊天、平淡剧情、无冲突章节。

## 情绪模型

压迫 -> 羞辱 -> 误解加深 -> 读者替主角憋屈 -> 主角亮底牌 -> 反派嘴硬 -> 更强证据打脸 -> 女主后悔或失控 -> 新危机压上来。

## 常用爆点

- 离婚当天身份曝光
- 被赶出家门后继承巨额资产
- 白月光当众翻车
- 女主悔到崩溃
- 反派以为赢了，下一秒跪下
- 主角隐藏身份被一点点揭开
- 极端偏爱：大人物只认主角
- 结尾钩子：真正幕后人出现，或女主发现更大的真相
